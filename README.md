## Installation

To use HAProxy script, simply download and run the `haproxy.sh` script. You can download the script from the following link:
```
bash <(curl -Ls --ipv4 https://github.com/Ramin-Setoodehnia/haproxy/raw/main/haproxy.sh)
```
